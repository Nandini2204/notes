5.1 Introduction:-एक graphical application programming interface है अर्थात यह एक GUI component है जो की windows मे GUI के लिए प्रयोग किया जाता है। विंडोज मे प्रदर्शित होने वाली सभी विंडो, प्रोग्रामिंग मे windows form कहलाती है। form एक control है जो की .Net मे windows interface provide करने के लिए प्रयोग किए जाते है। यह control other windows controls के लिए container का काम करता है। इसमे windows के सभी controlsको place कर windows applications के लिए User Interface तैयार करते है। Visual Basic के programs को runकरने पर display होने वाली window ही design time मे form कहलाती है। Form बनाने के लिए यूजर को ज्यादा मेहनत नहीं करनी पड़ती । यह built in functionality provideकरता है। form को design करने के लिए form designer window का प्रयोग किया जाता है। जो visual studio IDE का main component है।

Windows form मे Form का title bar, work area और border होता है जिसमे Title Bar मे Form Icon, Form’s Title, Control box जिसमे maximize, minimize और close के buttons होते हैं। इसके लिए windows form की properties होती हैं |जैसे- controlbox, icon, Text etc. Work area वह स्थान है जहां user controls को place कर form को designकरता है।

5.2 Window Forms:-Properties of Windows Form Name: form का name codes लिखते समय form को identify करने के लिए किया जाता है। इसे केवलdesign time मे change किया जा सकता है। Msgbox (Me.Name) Text: यह property form के title bar मे display होने वाले title को get या set करने के लिए use की जाती है। Me.Text = “Windows Form” AcceptButton: इस property का use form मे available किसी एक button को default set करने के लिए किया जाता है जो की ENTER key press करने पर automatically click हो जाती है। Me.AcceptButton = Button1 CancelButton: इस property का use form मे available किसी एक button को default set करने के लिए किया जाता है जो की Esc key press करने पर automatically click हो जाती है। Me.AcceptButton = Button2 BorderStyle: इस property का use form के border style को set करने के लिए किया जाता हैं। इसकी निम्न वैल्यू होती हैं- Size: इस property को form की size get या set करने के लिए प्रयोग किया जाता है। इसमे दो sub-properties height और width होती हैं। StartupPosition: इसका प्रयोग Form के run होने पर initial position को सेट करने के लिए किया जाता है। BackColor: इसका use form के background color को set या get करने के लिए किया जाता है। BackgroundImage: इसे Form के background मे image set या get करने के लिए किया जाता है। Font: इसे Font get या set करने के लिए प्रयोग किया जाता है। ForeColor: इसे Form के font color को सेट या get करने के लिए प्रयोग किया जाता है।

5.4Exception Handling:--> यह एक abnormal(असामान्य) या unexpected condition होती है जो program के execution के दौरान उत्पन्न होती है| जैसे किसी संख्या को शून्य से divide करने पर abnormal condition generate होती है जिसे DivideByZero Exception कहते है| 
-> जब भी exception उत्पन्न होता है तो कपिलेरेक exception object create करता है तथा exception से सम्बंधित सभी information object में store करता है| 
-> चूँकि exception run time पर उत्पन्न होते है इस कारण से हम इनको correct नहीं कर सकते है और हमारा program अचानक terminate हो जाता है| 
-> Program के अंदर run time पर उत्पन्न होने वाले exception में होने वाले नुक्सान को रोकने के लिए हमे program के अंदर Exception handler provide करना होता है| 
-> Exception handler एक mechanism होती है जो प्रोग्राम को execute करने का वैकल्पिक पाथ(alternative path) provide करती है| 
-> VB.Net में exception को handle करने के लिए exception नाम की class के different object create होते है तथा सभी प्रकार के exception Exception class से inherit होते है| जैसे DivideByZero तथा ArgumentException दोनों ही Exception class को inherit करके बनायी गयी है|

exception को handle करने के दो तरीके होते है| 1. Using try-catch-finally method
2. Using throw statement

catch(exception declaration { // the block of code that handle the exception
}
finally

2. using throw statement:-

user भी अपने स्वयं के exception को define कर सकते है इसके इसके लिए throw statement का उपयोग किया जाता है|
