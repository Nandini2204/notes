heap एक complete binary tree है जिसमें प्रत्येक parent node अपने child nodes की तुलना में अधिक या कम होता है। heap दो प्रकार के हो सकते हैं:

2. Min heap:-एक min heap एक tree है जिसमें प्रत्येक नोड का value अपने child node के value से कम या बराबर है।

6. Set PTR:=PAR
[End of step 2 loop]
7. [Assign VALUE as the root of H]
Set TREE[1]:=VALUE
8. Return


Procedure 2:-
DELHEAP(TREE,N,VALUE):
->A Heap H with N elements is stored in the array TREE.
->This procedure assigns the root TREE [1] of H to the variable VALUE and then reheaps the remaining elements.
->The variable LAST saves the value of the original last node of H.
->The pointer PTR, LEFT and RIGHT gives the location of LAST and its left and right children as LAST sinks in the tree.
1. Set VALUE:=TREE[1]
2. Set LAST:=TREE[N] and N :=N-1
3. Set PTR:=1,LEFT:=2 and RIGHT:=3

5. If LAST >=TREE[LEFT] and LAST>=TREE [RIGHT], then:
6. Set TREE[PTR]:=LAST and return

Set TREE[PTR]:=TREE[LEFT] and PTR:=LEFT
Else:
Set TREE[PTR]:=TREE[RIGHT] and PTR:=RIGHT
7. Set LEFT:=2*PTR and RIGHT:=LEFT+1
8. If LEFT=N and if LAST< TREE[LEFT],then: set PTR :=LEFT
9. Set TREE[PTR]:=LAST
10. Return

ऐसी situation जब data set U में दो या अधिक data elements के लिए resultant hashes होता है, उसी table के स्थान पर mapp करता है, इसे hash collision कहा जाता है। ऐसी situation में दो या अधिक data element hash element में उसी स्थान पर stored/mapped किए जाने के योग्य होंगे।
