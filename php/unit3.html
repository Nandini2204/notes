3.1 Usage of Arrays in Php:-Array एक Data Structure है जो कई elements या values को store कर सकता है| आसान भाषा में कहें तो Array एक special variable है जो कई सारी values को अपने अंदर hold कर सकता है|

ARRAY का SYNTAX:-

array(value1, value2, value3….)
अथवा
[value1, value2, value3….]

Array को represent करने के दो तरीके हैं और आप दोनों में से किसी को भी इस्तेमाल कर सकते हैं|
उदाहरण:–
PHP Code
1. < ?php
2. $a = array(“Hello”, “Pawan”, “Good Morning”);
3. print_r($a);
4. ?>
Array के लिए PHP ने एक inbuilt foreach loop provide की है जो ख़ास array के लिए ही है|

foreach loop का syntax:-

foreach(array_name as temporary_variable){
#Code
}
array_name – जिस array के लिए आप loop चलाना चाह रहे हैं, उसका नाम as – यह fix है, यह आपको लिखना ही लिखना है temporary_variable – एक temporary variable define कर दें उदाहरण:–
PHP Code
< ?php
$a = array(“Hello”, “Pawan”, “Good Morning”);
foreach($a as $b){
echo $b;
}
?>
Output
Hello
Pawan
Good Morning

2. Associative Array:-

Associative Array; Indexed Array जैसा ही होता है, लेकिन associative array में Programmer द्वारा index को define किया जाता है | index को define करने के लिए => इस sign के साथ value का इस्तेमाल किया जाता है | हर array के element को comma से seperate किया जाता है |
Syntax for Numeric Keys and Values
$arr1 = array( key 1 => value 1, key 2 => value 2, …., key n => value n );
Syntax for String Keys and Values
$arr2 = array( “key 1” => “value 1”, “key 2” => “value 2”, …., “key n” => “value n” );

3.3 Adding and Removing Items from Arrays:-

Adding items from arrays:-

1. Array_push():-

इस function का उपयोग किसी array के last में elements को add करने के लिए किया जाता है|
Example:-
< ?php
$fruits = array(“orange”,”banana”);
array_push($fruits,”apple”,”raspberry”);
print_r($fruits);
?>
Output:-
Array
(
[0]=orange
[1]=banana
[2]=apple
[3]=raspberry
)

2. array_unshift()function:-

इस फंक्शन का उपयोग किसी array के प्रारम्भ मे elements को add करने के लिए किया जाता है
Example:-
< ?php
$fruits = array(“orange”,”banana”); 
array_unshift($fruits,”apple”,”raspberry”);
print_r($fruits);
?>
Output:-
Array
(
[0]=apple
[1]=raspberry
[2]=orange
[3]=banana

Removing items from arrays:-

1.array_pop() function:-

इस function का उपयोग किसी array के last element को remove करने के लिए होता है
Example:-
< ?php
$fruits=array(“orange”,”banana”,”apple”,”rapberry”);
$fruits=array_pop($fruits);
print_r($fruits);
?>
Output:-
Array
(
[0]=orange
[1]=banana
[2]=apple
)

2. array_shift() function:-

इस function का उपयोग किसी array के प्रारम्भ से element को remove करने के लिए किया जाता है
Example:-
< ?php
$fruits=array(“orange”,”banana”,”apple”,”raspberry”);
$fruits=array_shift ($fruits);
Print_r ($fruits);
?>
Output:-
Array
\ (
[0]=banana
[1]=apple
[2]=raspberry
)


3.4 One – Dimensional and Multidimensional Arrays

ONE-DIMENSIONAL ARRAYS (1-D):-

वह arrays जिसमे सिर्फ एक subscript होती है उसे one dimensional arrays कहते है। इसका प्रयोग linear रूप में डेटा को स्टोर करने के लिए किया जाता है।

Multidimensional Arrays (M-D):-

वह arrays जिसमें दो से ज्यादा subscript होती है वह Muti-dimensional arrays कहलाता है।

3.5 Array Functions:–PHP में Array को handle करने के लिए कई सारे inbuilt function दिए गए हैं जिनकी मदद से हम कई बड़े कामों को बड़ी ही आसानी से कर सकते हैं| यहां हम आपको array से जुड़े कुछ जरुरी function के बारे में बताने जा रहे हैं –
count( ) – यह function array में कितनी values हैं ये count करता है|
print_r( ) – यह function array को print कराने का कार्य करता है|
sort( ) – यह function, Numeric Index Array के साथ use किया जाता है और यह array को ascending order में sort करता है|
rsort( ) – यह भी Numeric Index Array के साथ use किया जाता है और यह array को descending order में sort करता है|
array_push( ) – यह function, array में कोई नयी value insert कराने के लिए इस्तेमाल होता है|
array_pop( ) – यह function, array से last element को निकाल देता है|
min( ) – यह function, array की सभी values को चेक करके सबसे छोटी value को return करता है|
max( ) – यह array की सबसे बड़ी value को return करता है|
array_search( ) – यह किसी value को array में search करने के काम में आता है|
array_sum( ) – यह array की सभी values का sum करके return करता है|
array_rand( ) – यह array से कोई भी random value लेकर display करता है|
in_array( ) – इससे आप चेक कर सकते हैं कि कोई value array में exist है या फिर नहीं
array_unique( ) – यह array से सभी duplicate value हटाकर array को unique बनाता है


array_reverse( ) – यह array को reverse कर देता है
array_replace( ) – यह दो array को एक में replace कर देता है
array_diff( ) – यह दो दिए हुए arrays की values को चेक करता है कि उनमें क्या क्या different है
array_diff_assoc( ) – यह भी difference चेक करता है तथा associative array के साथ ही इस्तेमाल होता है|
array_diff_key( ) – यह दो दिए हुए arrays की keys को भी चेक करके difference बताता है|
asort( ) – ये function, associative array को ascending sort करने के काम में आता है| arsort( ) – यह associative array को descending order में sort करने के काम में आता है|
ksort( ) – यह Keys के आधार पर associative array को ascending sort करता है|
krsort( ) – यह Keys के आधार पर array को descending order में sort करता है|
sizeof( ) – यह same count function की तरह ही array का size return करता है|
array_keys( ) – यह array की सभी keys की लिस्ट return करता है|
array_key_exists( ) – यह function यह चेक करता है कि क्या दी हुई key, array में exist है या फिर नहीं|
array_product( ) – यह array के सभी values का multiply करके return करता है|
array_merge( ) – यह दो arrays को merge (एक को दूसरे में मिलाना) करने का
