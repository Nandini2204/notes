5.1 Introduction to JFC:-Swing JFC(Java Foundation class) का part है|Abstract windows toolkit की कामियों को पूरा करने के लिए java swing को बनाया गया है। AWT library basic GUI controls provide करती है। Java में swing AWT का advanced version है। AWT के साथ 2 problems होती है।
1.AWT programs size में बहुत बड़े होते है। 
2.AWT components fixed रहते है, आप उन्हें ज्यादा change नहीं कर सकते है। 
इन दोनों problems को swing remove कर देता है। Swing AWT पर based बना हुआ है। इसलिए swing AWT को replace नहीं करता है। यदि आप AWT के साथ familiar है तो swing को समझने में आपको कोई परेशानी नहीं होगी।

Features of Java Swing:-

1.सभी swing कंपोनेंट्स Platform independent होते है। 
2.सभी swing components lightweight होते है। 
3.Swing plug-gable look and feel को support करता है। 
4.Swing MVC(Model-View-Controller) architecture को फॉलो करता है। 

Java Swing Classes:-

1.JWindow:- JWindow container क्लास hierarchy को represent करती है। JFrame और JDialog क्लासेज इस class को higher करती है।


2.JFrame:- Java JFrame swing class AWT की frame क्लास के बेस पर बनायीं गयी है। ये एक container क्लास होती है। ये आपकी बेस window होती है जिसमे आप दूसरे components add करके अपनी एप्लीकेशन को design करते है। 
इसमें आप different components जैसे की buttons, labels आदि add कर सकते है। Components add करने के लिए आप add() मेथड यूज़ करते है।


3.JDialog:- इस क्लास का ऑब्जेक्ट क्रिएट करके आप अपनी एप्लीकेशन में Dialog box डाल सकते है।


4.JComponent:- ये क्लास swing hierarchy में सभी components को represent करती है। सभी swing components इस क्लास को directly inherit करती है।


5.JLabel:- इस क्लास का object क्रिएट करके आप text labels अपनी एप्लीकेशन में add कर सकते है।


6.JButton:- इस क्लास का object क्रिएट करके आप अपनी window में button ऐड कर सकते है। इसका ऑब्जेक्ट क्रिएट करते समय आप एक string पास करते है जो की बटन का नाम होती है।


7.JTextField:- इस क्लास का ऑब्जेक्ट create करके आप अपनी अपनी एप्लीकेशन में text box add कर सकते है|


8.JScrollBar:- ये class एप्लीकेशन में scroll bar ऐड करने के लिए यूज़ की जाती है।


9.JMenuBar:- इस क्लास के माध्यम से आप अपनी window में menu bar add कर सकते है।


10.JCheckBox:- इस क्लास का ऑब्जेक्ट क्रिएट करके आप अपनी window में check box add कर सकते है।


11.JList:- ये क्लास आपकी window में एक list ऐड करती है।


12.JTextArea:- इस क्लास के माध्यम से आप अपनी एप्लीकेशन में text area ऐड कर सकते है।


13.JRadioButton:- ये क्लास application में radio button add करने के लिए यूज़ की जाती है।


5.2 JApplets,JToolTip class:-JApplet एक java swing public class है जिसे developers के लिए design किया गया है जो आमतौर पर java में लिखा जाता है। JApplet आम तौर पर java bytecode के रूप में होता है जो सन माइक्रोसिस्टम्स से जावा वर्चुअल मशीन (JVM) या Applet viewer की मदद से चलता है। इसे पहली बार 1995 में पेश किया गया था।
JApplet को अन्य प्रोग्रामिंग भाषाओं में भी लिखा जा सकता है और बाद में इसे java byte code में compiled किया जा सकता है।

import javax.swing.*;

import java.awt.event.*; 
import java.awt.*; 
import javax.swing.*; 
class text extends JFrame implements ActionListener {
// JTextField 
static JTextField t; 
// JPasswodField 
static JPasswordField pass; 
// JFrame 
static JFrame f; 
// JButton 
static JButton b; 
// label to diaplay text
static JLabel l; 
// default constructor 
text() 
{ 
}
// main class 
public static void main(String[] args)
{ 
// create a new frame to stor text field and button 
f = new JFrame(“textfield”); 
// create a label to display text 
l = new JLabel(“nothing entered”);
// create a new button 
b = new JButton(“submit”); 
// create a object of the text class 
text te = new text(); 
// addActionListener to button 
b.addActionListener(te);
// create a object of JTextField with 16 columns and initial text 
t = new JTextField(“enter name”, 16); 
// create a object of passwodField with 16 columns
pass = new JPasswordField(16); 
// create an object of font type 
Font fo = new Font(“Serif”, Font.ITALIC, 20);
// set the font of the textfield 
t.setFont(fo); 
// create a panel to add buttons and textfield
JPanel p = new JPanel(); 
// add buttons and textfield to panel 
p.add(t); 
p.add(pass); 
p.add(b); 
p.add(l); 
// add panel to frame
f.add(p); 
// set the size of frame 
f.setSize(300, 300); 
f.show();
} 
// if the button is pressed 
public void actionPerformed(ActionEvent e) 
{ 
String s = e.getActionCommand();
if (s.equals(“submit”)) { 
// set the text of the label to the text of the field 
l.setText(“name = ” + t.getText() + “\t Password = ” + pass.getText());
// set the text of field to blank
t.setText(” “); 
// set the text of password field to blank 
pass.setText(“”);
} 
} 
}

JScrollPane:-

import java.awt.FlowLayout; 
import javax.swing.JFrame; 
import javax.swing.JScrollPane; 
import javax.swing.JtextArea; 
public class JScrollPaneExample {
private static final long serialVersionUID = 1L; 
private static void createAndShowGUI() { 
// Create and set up the window. 
final JFrame frame = new JFrame(“Scroll Pane Example”); 
// Display the window. 
frame.setSize(500, 500); 
frame.setVisible(true); 
frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); 
// set flow layout for the frame 
frame.getContentPane().setLayout(new FlowLayout()); 
JTextArea textArea = new JTextArea(20, 20); 
JScrollPane scrollableTextArea = new JScrollPane(textArea); 
scrollableTextArea.setHorizontalScrollBarPolicy(JScrollPane.HORIZONTAL_SCROLLBAR_ALWAYS); 
scrollableTextArea.setVerticalScrollBarPolicy(JScrollPane.VERTICAL_SCROLLBAR_ALWAYS); 
frame.getContentPane().add(scrollableTextArea); 
} 
public static void main(String[] args) { 
javax.swing.SwingUtilities.invokeLater(new Runnable() { 
public void run() { 
createAndShowGUI();
} 
}
);
}
}
