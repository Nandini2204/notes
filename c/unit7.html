2.istream(I/P stream)
3.ostream(O/P stream)

5.streanbuf


1.ios:-
->Contains basic facilities that are used by al other i/p and o/p classes
->Also contains a pointer to buffer object(streambuf objects)
->Declares constants and functions that are necessary for handling formatted input and output operation.


2. istream:-
->inherits the properties of ios
->Declares input function such as get(),getline()and read()
->Contains overloaded extraction operators>>


3.ostream:-
->inherits the properties of ios
->Declares output function such as put(),and write()
->Contains overloaded insertion operators>>


4. iostream:-inherits the properties of ios stream and ostream through multiple inheritance and thus contains all the input and output functions


5. streanbuf:-Provides an interface to physical devices through buffer
Acts as a base for filebuf class used ios files.
