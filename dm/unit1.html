1.1 Introduction:-Data mining को data या knowledge discovery भी कहते है। data mining, बहुत बड़े डेटा के समूह में से small डेटा को search करने की प्रक्रिया है। इस प्रक्रिया में परम्परागत statistics, artificial intelligence तथा computer graphics का प्रयोग किया जाता है। data mining में डेटा को analyze करने के लिए data mining tools का प्रयोग किया जाता है। ये tool बहुत ही powerful होते है।

.2 How data mining works:-

1. Data Cleaning:-

final analysis प्राप्त करने में डेटा का वर्गीकरण आवश्यक है। कोई भी डेटा जो incomplete, noisy और uncertian हो जाता है, परिणाम को प्रभावित कर सकता है। data cleaning एक record,table या database से मुश्किल या गलत डेटा को पहचानने और निकालने की प्रक्रिया है।

2.Data Integration:-

Data integration एक technique है जब हम मौजूदा जानकारी के साथ नई जानकारी का विलय करते हैं।sourcess में कई database,data cubes या flat files शामिल हो सकती हैं। data integration के one of the most कार्यान्वयन में से एक enterprise डेटा वेयरहाउस का निर्माण कर रहा है।

3. Data Transformation:-

डेटा या सूचना को एक प्रारूप से दूसरे में बदलने की प्रक्रिया को data transformation के रूप में जाना जाता है। यह आमतौर पर एक source system की संरचना से एक नई destination syetem की आवश्यक संरचना में किया जाता है। इस process में मूल रूप से documents को परिवर्तित करना शामिल है, लेकिन data conversions में कभी-कभी program को एक कंप्यूटर प्लेटफ़ॉर्म से दूसरे में परिवर्तित करने के लिए प्रोग्राम को अलग प्लेटफॉर्म पर चलाने के लिए authorise किया जाता है। इस data passage का उद्देश्य एक नई system को अपनाना है जो पिछले एक से बिल्कुल अलग है।

4. Data Discretisation:-

जिन techniques का उपयोग continuous attribute के domain को attribute में विभाजित करने के लिए किया जाता है, उन्हें data discretisation के रूप में जाना जाता है। विभिन्न अध्ययन विशेषता मान छोटे अंतराल लेबल द्वारा बहाल किए जाते हैं। इससे हमें mining results के ज्ञान स्तर के प्रतिनिधित्व का easy और compact ways से उपयोग करने में मदद मिलती है। 

5.Pattern Evaluation and Data Presentation:-

Data representation किसी भी business में एक महत्वपूर्ण भूमिका निभाता है। client या customer यदि किसी कार्यकुशल तरीके से प्रस्तुत किए जाते हैं, तो डेटा को सर्वश्रेष्ठ बना सकते हैं। एक बार जब डेटा उपरोक्त process से गुजर चुका होता है और मूल्यांकन किया जाता है कि यह त्रुटिपूर्ण है तो इसे diagrams और graphs के माध्यम से प्रस्तुत किया जाता है ताकि इसे न्यूनतम सांख्यिकीय(minimal statistical) ज्ञान वाले लोगों द्वारा समझा जा सके। 


1.3 Data Mining Tasks:-

1. Characterization:-

Data characterization ,target classses के data के general features का summerization है user के द्वारा specified class के data को database मे query के through collect किया जाता है तथा उस query के through acess भी कर सकते है OLAP ऑपरेशन्स के through हम data characterrization की process कर सकते है |

2. Discrimination:-

Data Discrimination डाटा rules को describe करती है | तथा यह दो classes के डाटा के बीच comparision करती है | जिनमे एक target class होती है | और दूसरी source class होता है यह classes user के through specified की जाती है | तथा इनके data को database queries के through retrieve कर सकते है | 

3. Association Analysis:-

Association analysis को association rules के through perform किया जाता है | इसके दिए गए डाटा पर condition apply की जाती है तथा result गेट किया जाता है | यह association more than one ऐट्रिब्यूट्स का association करती है | asssociation rules single dimensional और multidimensional होते है|

4. Classification:-

Classification analysis के given column के data का organization है | इसको surprised classification भी कहते है | इसमें classes मे स्टोर होने वाले डाटा का order मे classification किया जाता है classification class मे store data के collection पर training सेट apply करती है |

5. Prediction:-

Prediction किसी business के successfull नहीं होने से पहले ही की जाने वाली farecasting है | इसको mainly business के context मे ही use किया जाता है 


1.4 Data Mining Elements:-

Data Minnig consiste of five element:-

 1.डेटा वेयरहाउस सिस्टम पर लेनदेन, ट्रांसफ़ॉर्म और डेटा। 
2. Store और multidimensional database सिस्टम में डेटा का प्रबंधन।
3. व्यापार विश्लेषकों और सूचना प्रौद्योगिकी पेशेवरों के लिए डेटा पहुँच प्रदान करें।
4.आवेदन सॉफ्टवेयर द्वारा डेटा का विश्लेषण करें। 
5. डेटा को एक उपयोगी प्रारूप में, जैसे कि एक graph or table पर रखें|


1.5 Data mining architecture:-data mining architecture के बहुत सारें elements होते है जैसे:- data mining engine, pattern evaluation, data warehouse server, graphical user interface तथा knowledge base.

1.Data Sources:- database, डेटा वेयरहाउस, www (world wide web), text files तथा अन्य डॉक्यूमेंट, data के वास्तविक source होते है|data mining के लिए हमें बहुत बड़ी मात्रा में historical data की जरूरत होती है| organisation सामान्य रूप से data को databases या data warehouses में स्टोर करती है| data warehouse में हम एक से अधिक databases को स्टोर कर सकते हैं|
2. Database या Data Warehouse Server:- database या data warehouse server जो है वह actual data को स्टोर किये रहते हैं. यह data प्रोसेस होने के लिए तैयार रहता है और जब भी user कोई request करता है तो यह relevant data को retrieve करता है.
3.Data Mining Engine:-डेटा माइनिंग इंजन किसी भी data mining system का बुनियादी component होता है. इसमें बहुत सारें modules होते हैं जो कि data mining tasks जैसे:- association, classification, clustering, तथा prediction आदि को पूरा करते है.
4. Pattern Evaluation Modules:- pattern evaluation modules का मुख्य काम बहुत सारें patterns को find करना है. यह data mining engine के साथ interact करता है जिससे कि interesting patterns को खोजा जा सकें. 
5. Graphical User Interface:- graphical user interface module जो है वह यूजर तथा डेटा माइनिंग सिस्टम के साथ communicate करता है. यह module यूजर को सिस्टम को आसानी से तथा प्रभावी ढंग से use करने में मदद करता है. जब कोई user किसी सवाल या task को specify करता है तो यह module सिस्टम के साथ interact करता है और एक result प्रदान करता है.|
6. Knowledge Base:-knowledge base, डेटा माइनिंग की पूरी प्रक्रिया में helpful होता है. यह search को guide करने तथा interesting patterns को evaluate करने में बहुत उपयोगी साबित होता है. यह data mining engine को इनपुट प्रदान करता है जिससे वह patterns को search कर सकें.|


1.6 Data mining Advantages:-निम्नलिखित data mining advantage हैं:
Data mining से financial institutions और banks को probable defaulters की पहचान करने में मदद मिलती है और इसलिए उन्हें credit card, loan आदि जारी करने में मदद मिलेगी या नहीं। यह पिछले transactions, user के व्यवहार और data pattern के आधार पर किया जाता है|
यह advertisements को machine learning algorithms के आधार पर web pages पर इंटरनेट सर्फर के लिए सही विज्ञापन देने में मदद करता है। इस तरह से data mining दोनों संभावित खरीदारों के साथ-साथ various product के sellers को भी benifit होता है।
Retail malls और किराना स्टोर सबसे चौकस स्थिति में अधिकांश बिक्री योग्य वस्तुओं की व्यवस्था करते हैं और रखते हैं। data mining softwares से प्राप्त inputs के कारण यह संभव हो पाया है। इस तरह से data mining से राजस्व बढ़ाने में मदद मिलती है।
यह e-commerce वेबसाइटों (जैसे अमेजन, Taobao, अलीबाबा, स्नैपडील, वॉलमार्ट, फ्लिपकार्ट, बेस्टबाय, ईबे आदि) से संबंधित प्रश्नों के वांछित खोज परिणाम प्राप्त करने में मदद करता है, खोज इंजन (गूगल, याहू, बिंग, Ask.com DuckDuckGo आदि।)
यह कई अलग-अलग क्षेत्रों या डोमेन में उपयोग किया गया है। जैव-सूचना विज्ञान, चिकित्सा, आनुवंशिकी, शिक्षा, कृषि, कानून प्रवर्तन, ई-मार्केटिंग, इलेक्ट्रिकल पावर इंजीनियरिंग आदि। उदाहरण के लिए, आनुवंशिकी में यह व्यक्तियों के डीएनए अनुक्रम के आधार पर बीमारियों के जोखिम की भविष्यवाणी करने में मदद करता है।
यह ऊपर उल्लिखित कानून प्रवर्तन एजेंसियों द्वारा आपराधिक संदिग्धों की पहचान करने में मदद करता है|


1.7 Disadvantags:-

1. Privacy:- कंपनियों के लिए पैसे के लिए विभिन्न कंपनियों को अपने ग्राहकों की उपयोगी जानकारी बेचने की संभावना है। उदाहरण के लिए, अमेरिकन एक्सप्रेस ने अपने ग्राहकों द्वारा बनाई गई क्रेडिट कार्ड की खरीद को दूसरी कंपनी को बेच दिया है।
2. Security:- कई e-commerce कंपनियां इस बात से अवगत हैं कि किस समय में विभिन्न user ऐतिहासिक डेटा पैटर्न के आधार पर ऑनलाइन आते हैं। हमारी सुरक्षा के लिए उनके पास सुरक्षा व्यवस्था नही है।
3.Analyt: डेटा माइनिंग एनालिटिक्स सॉफ्टवेयर का संचालन करना मुश्किल होता है और इसके लिए उपयोगकर्ता को ज्ञान आधारित प्रशिक्षण की आवश्यकता होती है।
4.Different डाटा माइनिंग टूल्स अलग-अलग मैनर्स में काम करते हैं, क्योंकि उनके डिजाइन में अलग-अलग एल्गोरिदम होते हैं। इसलिए सही डेटा माइनिंग टूल का चयन एक थकाऊ और बोझिल काम है क्योंकि विभिन्न उपलब्ध उपकरणों के एल्गोरिदम, सुविधाओं आदि का ज्ञान प्राप्त करने की आवश्यकता होती है।
5.Can कंपनियों द्वारा डेटा माइनिंग के आधार पर प्राप्त जानकारी का दुरुपयोग लोगों के समूह के खिलाफ किया जा सकता है।
