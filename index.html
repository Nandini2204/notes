Data Structure

Data Structure इस तरह से data collecting और organising करने का एक तरीका है जिससे हम इन data पर effect तरीके से operation perform कर सकते हैं। data structure better organization और storage के लिए some relationship के rendering data structure in terms को प्रस्तुत करने के बारे में है। उदाहरण के लिए, हमारे पास कुछ data है, जिसमें player का नाम “virat” और age 26 है। यहां “virat” string data types का है और 26 integer data types का है। हम इस data को plyer record जैसे record के रूप में व्यवस्थित कर सकते हैं, जिसमें player का name और age दोनों होंगे। अब हम file या database में player के record को data structure के रूप में collect और store कर सकते हैं। उदाहरण के लिए: “Dhoni” 30, “Gambhir” 31, “Sehwag” 33 यदि आप object oriented programming concept के बारे में जानते हैं, तो एक class भी एक ही काम करता है, यह एक single entity के तहत विभिन्न प्रकार के data collect करता है। एकमात्र अंतर, data structures को manipulate data efficiently तक पहुंचने और access करने की technique प्रदान करता है।
